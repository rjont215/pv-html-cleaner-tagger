PhillyVoice HTML Cleaner & Tag Linker

Version 4.1 - Increased to 0-7 links + Fixed duplicate linking bug

What This Tool Does:

  • Removes all <span> tags (keeps content)
  • Removes problematic <div> tags with no attributes
  • Fixes related link packages (detects <hr> pairs with <150 chars containing links)
  • Extracts <img> tags from containing <p> and <h3> elements
  • Removes <br> tags after images
  • Removes empty <p> tags and <p> tags with only <br>
  • Removes specific inline style attributes from links
  • Converts smart/curly quotes to straight quotes
  • Normalizes em dash spacing (multiple spaces → single space)
  • Preserves all embed codes (YouTube, Instagram, Twitter, etc.)
  • NEW: Adds links to tag pages based on your Google Sheet (0-7 links max, first mention only)
